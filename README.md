# ApexPlanet Task 1 – Data Immersion & Wrangling

This repository contains my submission for **Task 1: Data Immersion & Wrangling** completed as part of the **ApexPlanet Data Analytics Internship**.

## Project Summary

The objective of this task was to acquire, inspect, clean, and prepare a raw dataset for analysis using Python and Pandas. The final output is an analysis-ready dataset along with the complete implementation notebook.

## Technologies Used

- Python
- Pandas
- Google Colab
- Microsoft Excel

## Repository Contents

- `Task1_Data_Cleaning.ipynb` – Complete data cleaning implementation.
- `ApexPlanet_DataAnalytics_Dataset.xlsx` – Original dataset.
- `Cleaned_Dataset.xlsx` – Cleaned dataset after preprocessing.

## Data Preparation Performed

- Imported and explored the dataset
- Examined data types and dataset structure
- Identified and removed duplicate records
- Handled missing values
- Generated descriptive statistics
- Exported the cleaned dataset for further analysis
